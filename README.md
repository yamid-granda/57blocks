## Min Requirements

- `Node.js >= @14 LTS` 
- `npm == @6` | OR | `pnpm == @7`

## Installation

With npm

```bash
npm install
```

With pnpm

```bash
pnpm i
```

## Run The Project in Dev Mode

With npm

```bash
npm run dev
```

With pnpm

```bash
pnpm dev
```

## Run Unit Tests

With npm

```bash
npm run test
```

With pnpm

```bash
pnpm test
```

## Run TDD (test driven development) Environment

With npm

```bash
npm run tdd
```

With pnpm

```bash
pnpm tdd
```
